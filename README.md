# Cognifyz Restaurant Data Analysis

## Overview

This project was completed as part of the Cognifyz Technologies Data Science Internship Program.

The objective was to analyze restaurant data, uncover business insights, perform exploratory data analysis (EDA), visualize trends, and build data-driven solutions using Python and data science techniques.

---

## Project Objectives

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Restaurant Rating Analysis
- Cuisine Analysis
- Geospatial Analysis
- Customer Preference Analysis
- Data Visualization
- Feature Engineering
- Predictive Modeling

---

## Dataset Features

The dataset contains information about:

- Restaurant Name
- City
- Country
- Cuisines
- Aggregate Rating
- Votes
- Price Range
- Online Delivery
- Table Booking
- Latitude & Longitude
